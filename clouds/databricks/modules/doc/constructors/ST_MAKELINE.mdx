## ST_MAKELINE

```sql:signature
carto.ST_MAKELINE(points)
```

**Description**

Creates a `LineString` using the given sequence of vertices in points.

* `points`: `Seq[Point]` input sequence of points for the line.

**Return type**

`LineString`

**Example**

```sql
SELECT carto.ST_MAKELINE(
  carto.ST_MAKEPOINT(-93.477736, 33.642527),
  carto.ST_MAKEPOINT(-93.47825, 33.642768)
);
-- NOT WORKING!
```
