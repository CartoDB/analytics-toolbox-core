## ST_Z2LATLON

```sql:signature
ST_Z2LATLON(geom, z2Index)
```

**Description**

Creates a `Z2Index` with the given coordinate Geometry.
Function used in the table optimization to generate a field to order the output table

* `geom`: `Geometry` input geom.

**Return type**

`Z2Index`

**Example**

```sql
SELECT carto.ST_Z2LATLON(carto.ST_GEOMFROMWKT("LINESTRING (0 0, 1 2)"));
-- {"min": "864691128455135232", "max": "864847779880401216"}
```
